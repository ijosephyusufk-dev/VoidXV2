Observer

Automatically detect and handle game events like player joins, deaths, and character spawns.

CreateObserver

local observer = Library.Developer:CreateObserver({
    OnPlayerJoined = function(player)
        print(player.Name .. " joined!")
    end,
    OnPlayerLeft = function(player)
        print(player.Name .. " left!")
    end,
    OnCharacterSpawned = function(player, character)
        print(player.Name .. " spawned")
    end,
    OnCharacterDied = function(player, character)
        print(player.Name .. " died")
    end,
    OnToolEquipped = function(player, tool)
        print(player.Name .. " equipped " .. tool.Name)
    end
})

Available Events

Event Parameters Description
OnPlayerJoined player Player joins the game
OnPlayerLeft player Player leaves the game
OnCharacterSpawned player, character Player's character spawns
OnCharacterDied player, character Player's character dies
OnToolEquipped player, tool Player equips a tool

Methods

observer:Destroy()  -- Stop observing and cleanup

Practical Example

local observer = Library.Developer:CreateObserver({
    OnPlayerJoined = function(player)
        Window:Notify("Player Joined", player.Name, 3, "info")
    end,
    OnCharacterDied = function(player)
        if player ~= game.Players.LocalPlayer then
            Window:Notify("Kill!", player.Name .. " died", 2, "success")
        end
    end
})
