ESP System

The ESP (Extra Sensory Perception) system provides powerful player visualization with Shadow ESP technology that covers the entire character.

üÜï New in v2.0: Shadow ESP uses Roblox's Highlight system for smooth, performance-optimized rendering.

Basic Usage

local esp = Library.Developer:CreateESP({
    Target = game.Players,
    Shadow = true,      -- Highlight-based ESP (covers entire character)
    Name = true,        -- Show player names
    Health = true,      -- Show health bars
    Distance = true,    -- Show distance
    TeamCheck = false,  -- Ignore teams
    MaxDistance = 500,  -- Maximum render distance
    Color = Color3.fromRGB(255, 100, 100)
})

Configuration Options

Option Type Default Description
Target Players game.Players The Players service to track
Shadow boolean true Enable Shadow ESP (Highlight)
Name boolean true Show player names
Health boolean false Show health bars
Distance boolean false Show distance in studs
TeamCheck boolean false Only show enemy team
MaxDistance number 1000 Maximum render distance
Color Color3 Red ESP color

Methods

Toggle Features

-- Toggle individual features (while ESP is active)
esp:ToggleShadow(true)    -- Toggle shadow glow
esp:ToggleName(true)      -- Toggle names
esp:ToggleHealth(true)    -- Toggle health bars
esp:ToggleDistance(true)  -- Toggle distance

Change Settings

esp:SetColor(Color3.fromRGB(0, 255, 0))  -- Change color
esp:SetMaxDistance(1000)                  -- Change max distance
esp:Toggle(false)                         -- Disable entire ESP

Cleanup

esp:Destroy()  -- Remove ESP completely

Complete Example

-- Create ESP
local esp = Library.Developer:CreateESP({
    Shadow = true,
    Name = true,
    Health = true,
    Distance = true,
    MaxDistance = 500,
    Color = Color3.fromRGB(255, 100, 100)
})

-- Create UI controls
MainTab:CreateToggle("Enable ESP", true, function(enabled)
    esp:Toggle(enabled)
end)

MainTab:CreateToggle("Show Shadow", true, function(enabled)
    esp:ToggleShadow(enabled)
end)

MainTab:CreateToggle("Show Health", true, function(enabled)
    esp:ToggleHealth(enabled)
end)

MainTab:CreateSlider("Max Distance", 100, 1000, 500, function(value)
    esp:SetMaxDistance(value)
end)

MainTab:CreateColorPicker("ESP Color", Color3.fromRGB(255, 100, 100), function(color)
    esp:SetColor(color)
end)
