Quick Start

Create your first VoidxV2 UI in under 5 minutes.

Basic Example

Here's a complete, working script that creates a simple UI:

-- Load the library
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ijosephyusufk-dev/VoidXV2/main/VoidLib.lua"))()

-- Create a window
local Window = Library:CreateWindow({
    Name = "My First Script",
    Subtitle = "v1.0.0",
    LoadingEnabled = true
})

-- Create a tab
local MainTab = Window:CreateTab("Main", "rbxassetid://7734053495")

-- Add a toggle
MainTab:CreateToggle("Auto Farm", false, function(state)
    print("Auto Farm:", state)
end)

-- Add a slider
MainTab:CreateSlider("Walk Speed", 16, 500, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

-- Add a button
MainTab:CreateButton("Print Hello", function()
    print("Hello World!")
    Window:Notify("Success", "Hello was printed!", 2, "success")
end)

Step by Step

1. Load the Library

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ijosephyusufk-dev/VoidXV2/main/VoidLib.lua"))()

This fetches and executes the VoidxV2 library, returning the main Library object.

2. Create a Window

local Window = Library:CreateWindow({
    Name = "My Script",        -- Window title
    Subtitle = "v1.0.0",       -- Optional subtitle
    LoadingEnabled = true      -- Show loading screen
})

The window is your main container. It includes the sidebar, tabs, and settings.

3. Create Tabs

local MainTab = Window:CreateTab("Main", "rbxassetid://7734053495")

Tabs organize your components. The second parameter is an optional icon.

4. Add Components

MainTab:CreateToggle("Name", false, function(state)
    -- Called when toggled
end)

MainTab:CreateSlider("Name", min, max, default, function(value)
    -- Called when value changes
end)

MainTab:CreateButton("Name", function()
    -- Called when clicked
end)
