Themes

Customize the appearance of VoidxV2 with custom themes.

Built-in Themes

Library:SetTheme("Dark")      -- Default dark theme
Library:SetTheme("Purple")    -- Purple accent
Library:SetTheme("Blue")      -- Blue accent
Library:SetTheme("Red")       -- Red accent
Library:SetTheme("Green")     -- Green accent

Custom Theme

Library:CreateTheme({
    Name = "MyTheme",
    Background = Color3.fromRGB(20, 20, 30),
    Sidebar = Color3.fromRGB(15, 15, 25),
    Accent = Color3.fromRGB(130, 80, 223),
    Text = Color3.fromRGB(255, 255, 255),
    SubText = Color3.fromRGB(160, 160, 160)
})

Library:SetTheme("MyTheme")

Theme Colors

Property Used For
Background Main window background
Sidebar Sidebar background
Accent Buttons, toggles, highlights
Text Primary text color
SubText Secondary/muted text

Live Theme Switching

-- Add theme selector to settings
Window:AddSettingsItem("Dropdown", "Theme", {"Dark", "Purple", "Blue"}, function(theme)
    Library:SetTheme(theme)
end)
