State Management

Save and restore component states and settings across sessions.

Using Flags

Flags are unique identifiers that connect UI components to saved states.

-- Create toggle with a flag
Tab:CreateToggle("Auto Farm", false, function(state)
    print("Auto Farm:", state)
end, "autoFarmToggle")  -- Flag name

-- Access the state later
local isEnabled = Library.Flags["autoFarmToggle"]

Save/Load Config

-- Save current config
Library:SaveConfig("MyScript")

-- Load saved config
Library:LoadConfig("MyScript")

-- Get available configs
local configs = Library:GetConfigs()
for _, name in pairs(configs) do
    print("Config:", name)
end

Auto-Save

-- Enable auto-save (saves on every change)
Library.AutoSave = true
Library.AutoSaveConfig = "MyScript"

Settings Tab Example

local SettingsTab = Window:CreateTab("Settings")

SettingsTab:CreateDropdown("Config", Library:GetConfigs(), function(config)
    Library:LoadConfig(config)
    Window:Notify("Loaded", "Config loaded: " .. config, 2, "success")
end)

SettingsTab:CreateInput("New Config", "Enter name...", function(name)
    Library:SaveConfig(name)
    Window:Notify("Saved", "Config saved: " .. name, 2, "success")
end)
