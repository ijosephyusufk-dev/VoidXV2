Installation

Get VoidxV2 up and running in your exploit script in seconds.

Requirements

  • A Roblox exploit with HTTP requests support
  • Script executor (Synapse X, Script-Ware, Fluxus, etc.)

Using Loadstring

The easiest way to use VoidxV2 is via loadstring. Simply copy and paste this at the top of your script:

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ijosephyusufk-dev/VoidXV2/main/VoidLib.lua"))()
üí° Tip: If you're experiencing caching issues, add a cache buster: 
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ijosephyusufk-dev/VoidXV2/main/VoidLib.lua?t=" .. os.time()))()

Verify Installation

To verify the library loaded correctly:

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ijosephyusufk-dev/VoidXV2/main/VoidLib.lua"))()

print("VoidxV2 Version:", Library:GetVersion())
print("Author:", Library.Author)
-- Output: VoidxV2 Version: 2.0.0
-- Output: Author: josefk
